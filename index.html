Git config

$ git config

Ver todas las configuraciones de git.

Git config list

$ git config --list

Se usa para verificar los cambios de configuracion que se han hecho.

Configura cuenta de github

$ git config --global user.name " "

Se emplea para ingresar el usuario de github.

Git config global email

$ git config --global user.email " "

Se emplea para ingresar el email de github.

Configura editor de código (vs code)

$ git config --global core.editor "code --wait"

Se utiliza para decirle al Git bash con qué editor trabajar.

Eliminar datos ingresados

$ git config --global --unset user.name

$ git config --global --unset user.email

$ git config --global --unset code.editor

Se utiliza para eliminar un dato.

dir

$ dir

Se utiliza para mostrar archivos.

pwd

$ pwd

Para mostrar en qué lugar de los discos estoy.

ls

$ ls

Enseña qué carpetas y archivos tengo abiertos.

Ubicación

$ cd c:

Se ubicara en el disco c.

Carpeta

$ cd repository

Se ubicará en la carpeta "repository" del disco c.

init

$ git init

Inicia una carpeta como repertorio de github

clone

$ git clone https://github.com/santiago-blip/SENA-GIT.git

Para descargar el archivo que tenga en el github, el https se saca del github.

status

$ git status

Para saber el estado del repositorio. Qué archivos se agregaron o eliminaron.

archivo a editar

$ cd sena-git

Para ubicarnos en el archivo a editar del git.

crear archivos

$ touch index.html/nombre del archivo

Crear un archivo de Git bash en vs code.

Abrir archivos

$ cd code "nombre del archivo"

Abrirá el archivo en el vs code.

Cambios

$ git add .

Para sincronizar cambios, va a agregar todos los cambios hechos.

comentarios

$ git commit -m " "

Este paso es obligatorio para poder enviar los datos, debe hacerse el comentario.

Enviar cambios

$ git push origin master

Subir archivos al github.

Actualizar

$ git pull

para actualizar lo de github al git bash.

Crear rama

$ git branch nombre-rama

Crear una rama del archivo master.

Ver ramas

$ git branch

Muestra todas las ramas creadas.

Cambiar rama

$ git checkout

Cambiarse de una rama a otra.
